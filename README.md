＃ 特征

- KernelSU ：KernelSU 是 Android GKI 设备的 root 解决方案，它以内核模式工作并直接在内核空间中向用户空间应用程序授予 root 权限。
- SUSFS ：KernelSU 的附加根隐藏内核补丁和用户空间模块。
